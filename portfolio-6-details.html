Dashboard Online Transaction Fraud Detection

Project Description

This project is an End-to-End Online Fraud Transactions project, aiming to create a pipeline from data collection to generating outputs in the form of a dashboard and notifications. The project includes a dashboard built with Looker, which provides real-time analytics and insights on transaction patterns, fraud detection, and prevention. The system is designed to detect and prevent fraudulent transactions, providing alerts and notifications to stakeholders, and ultimately reducing financial losses and improving customer trust.
